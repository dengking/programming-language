# 关于本章

本章总结C++ generic programming、template metaprogramming中解决一些常见问题的idiom/technique。之所以将它们放到同一个章节，因为在C++中它们的实现都是基于template的。

随着C++的发展，解决这些问题的idiom/technique也在不断发展，所以本章会以发展的视角总结解决这些问题的idiom/technique。

1、trait

2、sfinae

3、`enable_if`

4、tag dispatch

5、`void_t` C++17

6、concept C++20

7、constexp if C++17
