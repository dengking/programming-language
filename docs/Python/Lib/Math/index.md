## 数学分析

在 Python 生态中，虽然没有像 Mathematica 或 Maple 那样专为“符号数学分析”设计的单一商业软件，但通过组合使用多个开源库，完全可以实现强大的**数学分析（Mathematical Analysis）功能**，包括极限、导数、积分、级数、微分方程、泰勒展开等。

### SymPy —— 符号计算的核心库

**SymPy 是 Python 做数学分析的首选库**，完全用 Python 编写，支持：

- 极限（limit）
- 导数（diff）
- 不定/定积分（integrate）
- 泰勒展开（series）
- 微分方程求解（dsolve）
- 级数收敛性判断
- 多变量函数分析
- 拉格朗日乘子法、隐函数求导等



## 数值计算

当符号计算困难时，可用数值方法验证：

### NumPy + SciPy

当符号计算困难时，可用数值方法验证：

- `scipy.integrate.quad`：数值积分
- `scipy.misc.derivative`：数值导数
- `numpy.gradient`：离散梯度

> ⚠️ 注意：数值方法不能替代严格分析，但可用于验证或处理无法解析求解的问题。



## Python矩阵库

在 Python 中，处理矩阵（Matrix）最核心、最高效、最广泛使用的库是 **NumPy**。除此之外，还有其他一些库在特定场景下也非常有用。以下是主流的 Python 矩阵相关库及其特点：

### NumPy —— 矩阵计算的基石

**NumPy 是 Python 科学计算的基础库**，其 `ndarray`（N 维数组）对象被广泛用于表示向量、矩阵和高维张量。

### SciPy —— 高级线性代数与稀疏矩阵

基于 NumPy 构建，提供更专业的科学计算功能，尤其擅长：

- **稀疏矩阵**（Sparse Matrix）：`scipy.sparse`
- 大型线性方程组求解
- 特征值问题（如 `eigs` 用于大型稀疏矩阵）

### **SymPy —— 符号矩阵（用于解析推导）**

> 官网：https://www.sympy.org/

当你需要**符号化矩阵运算**（如含变量的矩阵、理论推导、教学演示）时，SymPy 非常强大。

### 其他补充库

| 库名                       | 用途                                  |
| ------------------------ | ----------------------------------- |
| **Pandas**               | 用 `DataFrame` 表示带标签的矩阵（适合数据分析，非纯数学） |
| **TensorFlow / PyTorch** | 深度学习框架，支持 GPU 加速的张量（含矩阵）运算，但主要用于 AI |
| **CuPy**                 | NumPy 的 GPU 加速版（需 NVIDIA GPU）       |

### 总结：如何选择？

| 需求场景                     | 推荐库                |
| ------------------------ | ------------------ |
| **通用数值矩阵计算**（科研、工程、机器学习） | ✅ **NumPy**        |
| **大型稀疏矩阵**（如图神经网络、有限元）   | ✅ **SciPy.sparse** |
| **符号推导、含变量的矩阵**（数学分析、教学） | ✅ **SymPy**        |
| **GPU 加速大规模计算**          | CuPy / PyTorch     |
| **带行列标签的数据表**            | Pandas             |
