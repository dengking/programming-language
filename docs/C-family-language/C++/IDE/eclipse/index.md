## [Eclipse IDE for C/C++ Developers](https://www.eclipse.org/downloads/)

### 如何使用doxygen风格的注释？

在Window->Preferences中搜索“doxygen"，然后会搜索到”C/C++“->"Editor"界面，将workspace default设置为doxygen。
