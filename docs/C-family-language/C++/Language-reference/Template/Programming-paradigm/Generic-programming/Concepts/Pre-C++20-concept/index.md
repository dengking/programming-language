# 关于本章

本章的标题"Pre C++20 concept"的含义是: 在C++20之前，如何使用类似于C++20 concept的用法来进行generic programming，下面是我目前已知的方式:

| 方式                                   | 章节                                            |
| -------------------------------------- | ----------------------------------------------- |
| The Boost Concept Check Library (BCCL) | `C++\Library\Boost\Boost-Concept-Check-Library` |
| ConceptC++                             | `./ConceptC++`                                  |
