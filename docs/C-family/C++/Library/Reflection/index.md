# Reflection

无论哪种实现方式，本质上都是在类型中添加描述其类型信息的meta-data，对于python类的语言，它的语言本身是携带有这些信息的，而对于c++这样的语言，语言并没有内置这些数据，所以需要由programmer来创建这些信息。
